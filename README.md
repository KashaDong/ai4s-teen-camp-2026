## 📌 项目简介 / About This Repository  
  
本仓库开源 AI4S Teen Camp 全部课程材料，包括：  
- 课程讲义、任务指南与拓展阅读  
- 交互式 HTML 教学页面  
- Notebook、代码与科学数据  
- 作业模板、项目展示资源  
  
📍该仓库是 **AI4S TEEN CAMP 学习驿站（2025-2026）** 官方配套资源库之一，支持学生营期学习与营后科研延展。  
  
## 🎯 课程理念 / Learning Philosophy  
  
科学的未来，属于懂AI的探索者。  
课程以 **“AI助力科学发现 / AI for Science”** 为核心，通过真实科学问题驱动学习：  
> 从看懂数据  
> → 理解模型  
> → 用AI发现规律  
  
培养学生科学家式的思维与探索能力。  
  
## 🧩 核心模块与任务主题 / Modules & Challenges  
  
学习驿站包含 AI4S 科学挑战营所有任务主题：  
1️⃣ **任务一：天气瓶中的数据科学**  
看穿天气瓶现象背后的数据处理！  
2️⃣ **任务二：神秘空间的引力规律**  
探寻深空之中看不见的力量法则！  
3️⃣ **任务三：数据感知之“神秘的豹猫”**  
走进AI模型训练和数据奥秘  
4️⃣ **任务四：AI的新发现（居家挑战）**  
开启一场AI创想之旅  
5️⃣ **任务五：递推数列中的隐藏规律**  
让代码发现数学序列的终极规律！  
6️⃣ **任务六：从生命科学到图像识别**  
体验AI如何“看见”与理解科学图像！  
7️⃣ **更多任务敬请期待**  
  
> 科学导师 & AI导师联合设计  
> 在真实科研语境中习得科学与智能融合思维
